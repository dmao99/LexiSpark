# LexiSpark
 
一款通过阅读场景进行英文单词记忆的纯前端小软件。
 
## 核心功能 
- **单词拼写练习**：选择场景或词典进行单词拼写训练  
- **上下文拼写**：在完整文章段落中练习单词的语境化记忆  
- **自动纠错**：自动标记拼写错误并记录高频错误单词
 
## 背单词的场景
- 📝 笔记本功能
- 📊 复习收藏单词和高频错词
- 📊 听写和复读训练
- 🔍 近义词和易混词对比测验
- ✏️ 易错单词改错训练
- 🖼️ 图背单词

## 英语词典
在resource/WordBooks目录中是收集到的英语词典，txt格式。
